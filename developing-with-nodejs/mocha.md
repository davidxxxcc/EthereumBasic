### Mocha

Mocha is a testing framework

### 單元測試（Unit Test）

以 function 為最小單位，驗證在特定情況下的 input 和 output 是否正確。

`mocha`的語法是這樣：

```
describe('測試標題', function(){

    it('測試內容', function(done){
        // 進行測試
    })
    it('測試內容2', function(done){
        // 進行測試
    })

})
```
